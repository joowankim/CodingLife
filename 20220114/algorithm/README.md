# 프로그래머스 - 짝지어 제거하기

문제 링크: https://programmers.co.kr/learn/courses/30/lessons/12973

## 문제 풀이 타임라인

- ~30분: 모르겠다. 진짜. 어떻게 해야되지 내 아이디어는 무조건 시간 초과인데
- ~40분: 해설 확인 후 구현 및 제출 - 성공

## 목표 결과

- 25분 내로 해결: 실패
- 1차 제출 성공: 성공
- 해설 보지 않기: 실패
